# 研报 HTML 输出（可选 · 自动出图）

> **默认产出 markdown 研报**（自然、轻、可直接读/存）。**只有用户要"HTML / 网页版 / 出图 / 分享版"时**，才按本 spec 生成自包含 HTML + 图。
> **有本地 content-to-html / data-slides 类 skill → 优先用它们**（视觉一致）；没有（多数会员）→ 用本 spec 的自包含 HTML，开箱即用。

## 产出什么
一个 `.html` 单文件（自包含，双击即看，无需服务器）：
- 头：研究对象 + 日期 + **数据来源标注**（哪个工具/是否会员数据）
- 研报正文（分块：标的=基本面/期权面/情绪/风险；板块=拥挤度五维）
- **自动图**（见下）
- **强制自我证伪结论**（最弱一环 + 置信度）+ 历史分布陈述
- 页脚 **930 免责**：客观数据 + 方法，不构成投资建议、无买卖方向

## 自动出图（按研究类型）
**标的研究**：
- IV 分位 / skew（柱或线）
- 期权链到期分布（柱）
- 关键数据卡（价格 / IV 分位 / 距财报 / 恐慌值）
- 历史分布图（类似信号配置后 N 天的 realized vol / 价格区间分布，有数据才画）

**板块 / 拥挤度研究**（对齐 `sector-crowding.md`）：
- **拥挤度仪表盘 gauge**（score 0-100 + 分档色：<40 绿 / 40-70 黄 / 70-85 橙 / >85 红）
- **五维雷达图**（D1-D5 各维得分）
- **成分股条形**（涨幅 / PE 分位——谁贡献拥挤、谁在分化）
- 拥挤度时间序列（score 历史 + 分档带，有数据才画）

## 技术实现（自包含 · 无私有依赖）
- **单 HTML 文件**，图用 **ECharts 或 Chart.js 的 CDN**（`<script src=...>`，具体 URL 以官方文档为准）；要完全离线可内联 SVG。
- 数据从研报取数结果填入，**数字照搬、标来源**；缺数据的图标"数据不足"，**不硬画**（数据真实性）。
- 样式：冷白底、简洁、**EdgeLab 中性品牌**（不带杰尼马/马奇诺/个人信息）。
- 存到 cwd 或用户指定路径，文件名如 `研报_<对象>_<日期>.html`。

## 930（图也守）
图只呈现**数据 / 结构 / 分位**，**不画"买卖点""目标价""该减仓"**。页脚固定免责。板块级图比个股方向图安全。
